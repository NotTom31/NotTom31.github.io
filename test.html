An Eternity Gone By (2024)

Trashfire Games - Gameplay Programmer
Engine: Unity 2021

An Eternity Gone By is a third-person hack-and-slash game with telekinesis abilities. As a gameplay programmer, I developed three unique enemy types, some notable features include patrolling, object pooling, and attack selection based on player positioning.
